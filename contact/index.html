    GNPS Workflow Contact information¶

    Last changed: {{ git_revision_date }}

    If you have any questions about how to use GNPS please first ask the community at the GNPS forum located here.

    In the event that there is an issue with the GNPS website or workflows, please contact the administrators at ccms-web@cs.ucsd.edu.

    For more in-depth assistance, we invite you to attend the numerous GNPS workshops we offer at UCSD and at conferences throughout the year. Check out the upcoming workshops.
